# 广告总结

  1. 训练营最后一天8点到8点5分之间 很多人问到课程 没有一一的回复 在这里把课程再说一下 然后提醒一下最后一天 前三名 送机械键盘 （一定强调最后一天）
  2. 对比
  • 自学 vs 老师带领
  • 自学跟不上前端更新的速度 ， 我们的课程是最新的课程 并且在不断的更新
  • 自学的话遇到了问题会卡半天会丧失信心 老师指导你一下 轻松很多
  • 工作中的问题 bug 遇到了 也是可以问我们老师 来解决的
  • 每个入学的同学 都会根据你的情况制定学习计划 让你少走弯路
  1. 跳槽
  • 金九银十 马上就来了 面试就是一场打仗 最低也要提前一个月来准备
  • 现在加入进来刚好为九月份的面试做准备 老师也会针对你的加入制定面试的计划
  1. 时间的问题
  • 加入我们的90%都是有工作的 但是我们只是在工作中的知识很有限 这就导致了有很多前端的知识我们都落下了 距离我们的职业规划 距离大前端越来越远
  • 我们的课程是周二 周三 周四 夜晚8点 到 10点 开始 这个时间你刚好可以来听 也不需要你练习 可以利用周末的时间来提升 我们会安排作业 周末你来完成 作业 我们老师都是会一对一批改的
  1. 现在刚好3期即将结束 4期即将开始 现在加入进来 三期的所有资料 录播都是免费提供的 等四期开始了 三期的资料就获取不到了 这个也是腾讯课堂规定的 所以说 现在的时机是非常适合的
  2. 亮出我们的课程 突出 8个企业级的实战项目 才 3000多元 很划算
  3. 课程价钱很便宜 平时你遇到问题请别人解决 请别人吃顿饭都要几百元 我们学一年的话 一天就十几块钱 也就是少抽一包烟的钱
  4. 花了钱 你也有一个自我督促 你会觉得自己花了钱 不学浪费了 所以这也会督促你 公开课免费的 听一年的作用不大

# 广告技巧

  • 从前端如何涨薪资的话题引出vip的学员找到工作offer的截图 增加说服力度
  • 开发了两三年 没有系统学习vue 没有系统学习react 此时跳槽没有多少底气 虽然jquery用的熟悉 但是相信 在开发中也是找各种的插件 对原生js的高级了解也不是很深 这种同学就非常适合我们的课程 课程价钱也不贵 你也可以立马听的懂 找到工作后没有底气 接了项目 老师也是可以给你答疑的 夜晚抽出时间来听课程的话 可以很快的学会 并且我们还有 js高级进阶 框架不仅教你会用 还会剖析框架的原理
  1. 有没有第一次来听课的(这个在课程开始之前说 引导加微信)
  2. 大学生还没有招工作的 想转行的 我们是有基础的视频给到你学习的 你在线下现在也有很多是视频班级 最需要的就是有人能够指导 我们的老师在线指导的 此时把我们的老师阵容拿出来
  3. 刚在线下培训完 工作1-5年的时间 工作中做的事情就是代码的搬运 不敢跳槽 跳槽了工资提不上去的这一部分同学 我们主要会在js的高级进阶 js的工程化透过js的高级进阶 来分析框架的底层原理 js的设计模式 等等 来助力你在前端提升一个档次
  4. 我们是一家用心做教育的机构 给同学们看看答疑的服务 找到offer的截图
  5. 618刚过去 端午节我们都是有很大的优惠活动的 此时把价格亮出来 把优惠亮出来
  1. 不要把时间浪费在百度上面
  2. 仔细的介绍课表
