# 作用域执行期上下文的理解

当函数执行时，会创建一个称为执行期上下文的内部对象（AO）。一个执行期上下文定义了一个函数执行时的环境， 函数每次执行时对应的执行上下文都是独一无二的，所以多次调用一个函数会导致创建多个执行上下文，当函数执行完毕， 执行上下文被销毁 全局的是会创建一个全局的执行期上下文 称作是（GO）

# [[scope]]

中文名称 域的意思 js程序运行过程中有的属性我们可以访问的到 有的我们访问不到 供程序在执行的时候自己内部用 [[scope]] 就是其中一个 这个访问不到的scope的里面存储的就是 作用域链
