# 记录和她在一起的点滴

  1. 夜晚讲完课 她准备好的一个鸡腿一个鸡肉卷 吃完真的是好撑啊 然后两个人还骑车出去逛了逛 在路上那个傻子坐在车子的后面突然把口罩拿出来放在我的右边 当时正在骑车子的 吓得我一大跳 最后放一张她的美照吧 看看这大长腿
  2. 疫情期间整整四十天没有见面 第一次见面 我又没忍住打球了 她等的太久了 日常生气了 从那次后 我心里想 以后不能再这么自私了 来两张照片纪念一下
  3. 那一次去新西亚超市 门口的跷跷板 我和她都想坐 但是无奈被小朋友占住了 都出了门口 两个人还是回去了 是你让小朋友让座的 哈哈哈 那也是你为数不多的主动 可能你看出来我实在不好意思 去和小朋友打招呼 当然那美好的瞬间 不能不记录下来 你看看你笑的多么的开心
  4. 其实最难忘的就是疫情期间 解封了之后 每次下午的一点多我都会准时出现在三中门口 那段时光真的非常的令人怀念 不知道你是否也一样 县城本来也不到 盛湾公园应该是每次基本都会去的地方 那当然也少不了给我大宝宝拍照 就是没次好像技术都要被吐槽
  5. 还有一个地方不得不提的就是森林公园 在那儿给你拍了一张我最满意的图片 那一次的记忆最深刻的就是 把你抱起来转圈圈了吧 两个人笑的非常开心
  6. 除了森林公园 还有就是龙山水库了 那一天的天气特别好 不知道你还记不记得
  7. 五月三号我们一起来到了武汉 你没有想太多就陪伴着我一起来了 我们一起从武汉站坐的地铁 记得武汉站基本没人 我两个的心里还是挺恐慌的 来到武汉之后 我们一起去找房子 累了一天 你在地铁上都快累的睡着了 到了地铁口我们又等公交车 两个人满脸的油 虽然很累 但是真的是满满的幸福感

程颖还记得我一直说过吧 我们之间快乐的时光非常多，每次你闹脾气，就是觉得我没有以前那么在乎你了 我没有以前那么爱你了其实真的不是，我比以前更要爱你，有的时候我确实和你谈话的时候心不在焉，但是真的没有不在乎你，希望你以后少生点气，毕竟老是生气对身体不好。其实除了这些，真的从内心里面来讲感谢你。感谢你每天给我做饭，给我洗衣服。感谢你每天把家里收拾的那么干净，其实这些活很累。感谢你不嫌弃我，做什么事情都支持我，都愿意陪我一起。感谢你每次像小猫咪一样的粘着我，那也是我最幸福的时候。感谢你，让我觉得我不再是一个人，谢谢你！以后的日子里，我还是希望我们可以开心度过每一天！我们都要一起加油！！
